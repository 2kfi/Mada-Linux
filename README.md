# Mada Linux

**Mada Linux** is an Arch-based distribution meticulously crafted to bridge the gap between high-performance computing and native Arabic localization. Built for users who demand a minimalist, polished aesthetic without sacrificing the power and flexibility of the Arch ecosystem.

---

## 🎨 Philosophy
Mada (مدى) represents "horizon" or "scope." The project is centered on three core pillars:
* **Native Arabic Integration:** Out-of-the-box support for Arabic typography, right-to-left (RTL) UI optimizations, and localized tooling.
* **Minimalist Design:** A clean, modern interface inspired by high-end industrial design, focusing on clarity and intentionality.
* **Power & Control:** Leveraging the Arch Linux base to provide a rolling-release system that is lean, fast, and fully customizable.

---

## 📦 The Five Flavors
Mada Linux is offered in five distinct editions to suit every workflow and hardware configuration:

* **Mada Base:** The foundation. A GUI-less environment for servers, minimalists, or those who want to build their desktop from the ground up.
* **Mada KDE:** The ideal transition. A feature-rich, polished environment designed specifically for users migrating from Windows who want familiarity paired with Arch's power.
* **Mada GNOME:** The touch-ready edition. Tailored for tablet and mobile users, featuring a gesture-centric workflow and a clean, modern interface.
* **Mada Cinnamon:** The classic choice. Provides a traditional, Windows-like desktop experience that remains lightweight and efficient on system resources.
* **Mada Hyprland:** The flagship for enthusiasts. A dynamic Wayland tiling compositor designed for advanced users who want maximum control, keyboard-driven efficiency, and high-end "ricing" capabilities.

---

## ✨ Key Features
* **Modern Graphics:** Native Wayland support across all variants for tear-free rendering and superior multi-monitor handling (optimized for 2K/1440p displays).
* **Arabic Excellence:** High-quality font rendering and system-wide RTL support that feels integrated, not added-on.
* **Performance First:** Custom kernel hooks and optimized binaries to ensure low-latency performance, even on resource-constrained hardware.
* **The "Mada" Build Tool:** A custom ISO builder that allows users to spin their own versions of the OS with specific package sets.

---

## 🚀 Getting Started

### Prerequisites
* An x86_64 compatible processor.
* A minimum of 4GB RAM (Mada Base and Cinnamon can run on less; 8GB recommended for KDE/GNOME).
* A stable internet connection for the installation process.

### Installation
1.  Download the ISO for your preferred flavor from the [Releases](https://github.com/2kfi/Mada-Linux/releases) page.
2.  Flash the ISO to a USB drive using `dd` or your preferred flashing tool.
3.  Boot into the live environment.
4.  Run the `mada-install` script or system-install app to begin the guided setup.
